# HTML5

## SEMÂNTICA

<section> - seção genérica
<header> - Cabeçalho
<article> - conteudo principal
<aside> - conteudo relacionado ao principal (biografia)
<footer> - rodapé
<h1> ao <h6> - representam a importância do título em uma página

Só pode haver 1 h1 por página

## TAGS UTILIZADAS

Textos
<p>Conteúdo do artigo.</p>

Links
<a>Link</a>

href - link para onde a âncora está apontando
target - indica como esse link vai ser aberto
_blank - diz para a página abrir o link em uma nova aba

Imagens

<img>
<img src="img/avatar.jpg"> o source é obrigatório e ele guarda o caminho da imagem
<img alt="Foto de Lucas"> não é obrigatório mas é recomendado para maior acessibilidade, mostra a descrição da foto quando não é carregada.

Listas

<ul> - representa uma lista onde a ordem dos itens não é importante
<ol> - a ordem dos itens é importante
<li> - item na lista

FONTES
Web Save Fonts - Estão salvas na maioria dos dispositivos
Add 2 fontes para se a 1ª não funcionar entrar em campo a 2ª
font_family: 

font-size: altera o tamanho do texto.
font-style: altera a aparência do texto (normal, itálico)
font-wheight: altera o peso da fonte (normal, bold - negrito)
text-transform: altera texto entre maiusculo e minusculo
uppercase - CAIXA ALTA
lowercase - caixa baixa
capitalize - Todas As Primeiras Letras Maiusculas
text-dedcoration: coloca linhas
underline - linha abaixo
overline - linha acima
line-trought - linha ao centro cortando a palavra
